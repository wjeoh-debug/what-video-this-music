## 🚀 기능 목록

### [F1] 영상 그리드 목록 화면
> **Status**: 🔄 진행 중
> **목표**: 음악을 BGM으로 쓴 숏폼 영상들을 탭 필터와 함께 그리드로 보여준다

- [ ] 목업 샘플 데이터 準備 (유튜브/틱톡/릴스 각 3~5개씩)
- [ ] 곡 정보 헤더 영역 만들기 (제목, 아티스트명, 앨범 아트)
- [ ] 상단 탭 필터 만들기 (`전체 | 유튜브 | 틱톡 | 릴스`)
- [ ] 2열 영상 카드 그리드 레이아웃 만들기 (썸네일, 플랫폼 뱃지, 조회수)
- [ ] 탭 선택 시 해당 플랫폼 영상만 필터링되도록 동작 구현

### [F2] 인앱 영상 재생 화면
> **Status**: ⏳ 예정
> **목표**: 카드를 탭하면 앱 안에서 바로 영상을 재생하고, 원본 플랫폼으로도 이동할 수 있다

- [ ] 영상 재생 화면(모달 또는 새 페이지) 레이아웃 만들기
- [ ] YouTube embed 플레이어 연동 (유튜브 영상용)
- [ ] TikTok embed 플레이어 연동 (틱톡 영상용)
- [ ] "해당 플랫폼에서 보기" 버튼 추가 (클릭 시 새 창으로 오픈)
- [ ] 닫기 버튼으로 목록 화면으로 돌아오기
