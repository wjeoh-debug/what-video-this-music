## 프로젝트 목표

FLO 앱에서 특정 노래를 배경음악으로 사용한 유튜브 쇼츠, 틱톡, 인스타그램 릴스 영상을 한 곳에서 탐색하고 감상할 수 있는 인앱 웹뷰 기능

## 타겟 사용자

FLO 앱 사용자 — 듣고 있는 음악이 숏폼 크리에이터들에게 어떻게 활용되는지 궁금한 누구나

## 주요 기능

### 기능 1 — 영상 그리드 목록

- 설명: 특정 곡의 더보기 메뉴 → "이 음악이 사용된 영상 보기" 버튼 → 웹뷰 진입
- 상세:
  - 상단에 현재 곡 정보(제목, 아티스트) 표시
  - 해당 음악을 BGM으로 사용한 숏폼 영상들을 2열 그리드로 표시
  - 각 카드: 썸네일 + 플랫폼 뱃지(유튜브/틱톡/릴스) + 조회수
  - 상단 탭 필터: `전체 | 유튜브 | 틱톡 | 릴스`

### 기능 2 — 인앱 영상 재생

- 설명: 영상 카드 탭 → 웹뷰 내 플레이어에서 영상 재생
- 상세:
  - YouTube Shorts, TikTok 등 embed 플레이어로 인앱 재생
  - 플레이어 하단에 "유튜브/틱톡/인스타에서 보기" 버튼
  - 버튼 클릭 시 해당 플랫폼 링크를 새 창으로 오픈

### 기능 3 — (2단계) 실제 API 연동

- 설명: 실제 각 플랫폼 API를 통해 영상 목록 실시간 조회
- 상세:
  - YouTube Data API v3
  - TikTok Display API
  - Instagram Graph API
  - 1단계(현재)는 하드코딩된 샘플 데이터로 목업 구현

## 사용 환경

- 플랫폼: FLO 앱 인앱 WebView
- 로그인 필요 여부: 불필요 (FLO 기존 로그인 상태 그대로 사용)

## 데이터 저장

- Supabase 사용: 미사용
- 저장할 데이터: 없음 (외부 API 실시간 조회 / 1단계는 목업 데이터)

## 외부 연동

- 1단계(현재): 없음 (목업 하드코딩 데이터)
- 2단계(추후): YouTube Data API v3, TikTok Display API, Instagram Graph API

## 이번 버전에서 하지 않는 것 (스코프 아웃)

- 실제 API 연동 (목업 먼저, API 연동은 2단계)
- 영상 검색/정렬 기능
- 영상 저장/공유 기능
- 관리자 페이지
